Body Tag with Attributes



The Body Tag (<body>) can have several attributes that you can use to define the formatting features for the entire body of the HTML document. Some of the important attributes are listed in the following table:


Attribute Description Syntax
bgcolor You can give attractive colours to the background of your web page by using the bgcolor attribute. body bgcolor="color name"
text The text attributes is used to give colour to the body text. This attribute will colour all the text in the HTML page in one colour. body text="colour name"
background You can also add a picture in the background of a web page.
*You cannot use the background and bgcolor attributes together.		 body background="path of the image file to be set as background"



Example : <body bgcolor = "white" text = "green">
