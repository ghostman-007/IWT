Bold Tag


The Bold tag (<b>) specifies that the enclosed text should appear in bold face.

Syntax : <b> text </b>


Italics Tag


The Italics tab (<i>) indicates that the enclosed text should appear in italics.

Syntax : <i> text </i>


Underline Tag


The Underline tag (<u>) indicates that the enclosed text should be underlined.

Syntax : <u> text </u>
