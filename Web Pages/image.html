Adding Image to Web Page


We can add images to a web page in two ways:
  1. In the background of a web page, using the 'background' attribute of the body tag.
  2. In the foreground on a web page, using the 'src' attribute of the img tag.

Adding an Image in the background The background attribute of the body tags lets you add an image in the background of a web page.
Syntax : body background = "Location of the file"

Adding an Image in the Foreground To set an image in the foreground, we can use the img tag with the 'src' attribute. This location of the image is specified with the src attribute. If the image file is in the same folder as the HTML document, we need to give only the file name with the src attribute.
Syntax : <img src="Location of the file">

Tag/Attribute Description Syntax
src This attribute is used to specify the location and name of the image file. <img src="location of the file">
align This attribute is used to specify the position of image in the web page. It can have 6 positions which are top, middle, bottom, left, center, right. <img align=top>
border This attribute is used to specify the border around the image. <img border=4>
alt This attribute is used to indicate the text to be displayed in case the browser is unable to display the image specified in the src attribute. <img alt="File not found">
