Ordered List


An Ordered List is a sequential list where each list item is preceded by a number (or a letter).
An ordered list is also known as the Numbered list. The syntax for creating an ordered list is identical to that of creating an unordered list, except that it uses the <ol> tag. The items for an ordered list are tagged using the same <li> tag.

Syntax :<ol>
<li> list item 1
<li> list item 2
...
/ol>

Using the "type" Attribute To Change the Bullet Type
We can use the type attribute with the ol tag to change the numbering style.
The type attribute in ol can have values as shown below:
  1. <ol type="A"> ,creates ordered list of capital letters.
  2. <ol type="a"> ,creates ordered list of small letters.
  3. <ol type="I"> ,creates ordered list of upper Roman letters.
  4. <ol type="i"> ,creates ordered list of lower Roman letters.
  5. <ol type="1"> ,creates ordered list of arabic numerals (DEFAULT).

    Syntax :<ol type="1">
    <li> list item 1
    <li> list item 2
    ...
    </ol>
