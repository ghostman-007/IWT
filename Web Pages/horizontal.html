Horizontal Tag

The Horizontal Tag (<hr>) is an empty tag which is used to draw a horizontal line on the web page to highlight a certain item or to seperate two seconds of a web page. The <hr> tag has attributes that allow us to define the size, width, alignment and shading for the horizontal line that you want to insert. The following table lists these attributes:

Tag/Attribute Description Syntax
<hr> tag Lets you insert a horizontal line on the web page. <hr>
size Lets you define the thickness of the horizontal line in pixels. <hr size=6>
width Lets you define the length of the horizontal line as a percentage with respects to the entire screen. <hr size=6>
width=80%
align Lets you align the horizontal line left, right or center. <hr size=6 width=80% align=center>
color Lets you give a colour to the horizontal line. <hr size=6 width=80% align=center color=red>
