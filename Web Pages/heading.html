Heading Tag


The Heading Tag (<h>) is a container tag used to spocify the heading in a web page. A HTML document can have six different levels or sizes of headings - h1 to h6. A smaller number suffixed after h gives you a bigger sized heading.

Syntax :<hn> heading text </hn>
Where n is the size of the heading text which can take any value between 1 to 6.

Example :

Heading 1 i.e h1

Heading 2 i.e h2

Heading 3 i.e h3

Heading 4 i.e h4

Heading 5 i.e h5
Heading 6 i.e h6
