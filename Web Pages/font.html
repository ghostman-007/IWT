Font Tag


The Font Tag (<font>) is used to format a block of text. We can change the font type, size, colour of the text in a paragraph with the help of its attributes.

Tag/Attribute Description Syntax
face This attribute is used to specify the font type for the text. <font face="comic sans ms">
color This attribute is used to specify the font colour for the text. <font face=comic sans ms" color="red">
size This attribute is used to specify the font size for the text. The font is specified in number ranging from 1-7. 1 is smallest font size and 7 is the largest. <font size=5>
