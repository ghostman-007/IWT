Unordered List



An Unordered List (also known as Unnumbered list) is a list consisting of the list items preceded by bullet marks.
An unordered list starts with the tag (ul stands for unordered list). The list items are placed after the <ul> tag with each item preceded by the <li> tag (li stands for list item). The list is closed by using the tag.

Syntax :<ul>
<li> list item 1
<li> list item 2
<li> list item 3
...
</ul>

Using the "type" Attribute To Change the Bullet Type
We can use the type attribute with the ul tag to specify which bullet symbol we want to have preceding the items in an unordered list. The syntax for using "type" attributes is as follows:
The disc, circle and square ar values that the type attribute can have.

Syntax :<ul type = "bullet">
<li> list item 1
<li> list item 2
<li> list item 3
...
</ul>
