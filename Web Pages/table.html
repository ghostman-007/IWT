Table Tag



Tables are used on a web page to organize data in the form of horizontal rows and vertical columns. The use of tables simplifies complex and bulky data making.
Each table is made up of a fixed number of rows and columns. The rectangular are formed at the intersecton of a row and a column is called a cell. We enter data in the cells. Mostly a table contains a header row which specifies what items should come in the columns below.

The <Table> Tag

In HTML tables are created by using the tag. The <table> tag is a container tag therefore you we close it by using a colsing tag . Everthing within a table is enclosed between these two 'tags' which are both required.

Syntax : <table>........</table>
The <table tag has following attributes:

Attribute Use Example
align Specifies the horizontal position of the table within the web page window. The value of this attribute may be left, center or right. <table align=left>
width Controls the width of the entire table. <table width=100%>
cellspacing Adds space between the cells. The spacing between cells is given in pixils. <table cellspacing=20>
cellpadding Adds space around cell contents. <table cellpadding=10
valign Lets you align the cell contents in the table to the top or to the bottom in the cells. <table valign=top/bottom>
border Gives a border to a table. Border is thickness specified in pixils. <table border=2>
bordercolor Adds a colour to the border of a row or a cell. <table bordercolor=navy>
bgcolor Adds a colour to the background of a row or cell. <table bgcolor=yellow>


The <tr>, <th> and <td> tags

The table tag itself is of no use unless the <tr>, <th> and <td> tags are used to create other elements of the table.
  1. The <tr> tag creates a new row in the table. It is a container tag and needs to be closed with the </tr> tag.
  2. The <th> tag is used within the <tr> tag and creates a header cell of the table. It is also a container tag and needs to be closed with the </th> tag. The header cell data is enclosed within these two tags.
  3. The <td> tag is also used within the <tr> tag and that is to appear in a data cell.


The Attributes of <td> and <th> tags



Attribute Use Example
rowspan This attribute allows us to extend a row vertically by a given number of cells. <td rowspan=3> will extend the current row down to include the next three cells.
colspan This attribute allows us to extend a columns horizontal by a given number of cells. <td colspan=3> will extend the current column rightwards to include the next three cells.
