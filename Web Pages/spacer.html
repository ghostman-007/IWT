Spacer Tag



This Spacer Tag (<spacer>) is used to for inserting blank spaces in an HTML web page.

It has various attributes:

  • TYPE => Horizontal or Vertical.
  • SIZE => It indicates the amount of space to be left.
    Size accept only integer value.

Example : <SPACER type="Horizontal" size=10>

NOTE : The spacer command is understood by the Netscpae Navigator.
