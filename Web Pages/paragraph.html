Paragraph Tag

The Paragraph Tag (<p>) is a container tag used to create paragraph in a HTML document. The ON tag (<p>) marks the beginning of a paragraph and the OFF (</p>) tag marks the end of a paragraph.

Syntax :<p> text or picture to be converted into paragraph.</p>

The align attribute of the paragraph tag allows you to align a paragraph in three different ways on the web page - Left, Right and Center.


Attribute Description Syntax
align left aligns the paragraph from the left edge of the web page <p align="left">
align right aligns the paragraph from the right edge of the web page <p align="right">
align center aligns the paragraph from the center of the web page <p align="center">
